# March-Madness-Analysis
The Python script contained in this repository can be used for obtaining and formatting statistics obtained from kenpom.com. The R script can be used for some exploratory modeling and analysis with this data.Due to TOS, datasets cannot be distributed along with this code.

This code is provided for educational purposes only. Users are responsible for complying with the Terms of Use from the data sources, as found [here](https://kenpom.com/terms.php).
